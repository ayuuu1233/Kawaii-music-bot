🎧 Kawaii Music Bot 💖

«✨ A Cute Anime-Themed Telegram Music Bot with Voice Chat Streaming, Queue System & Kawaii UI 🎶»

---

🌸 Features

💖 Beautiful kawaii-style interface
🎧 Play music directly in Telegram voice chats
📜 Smart queue system (auto next)
⏯️ Controls: Pause, Resume, Skip, Stop
🔍 YouTube search support
🎀 Cute startup animation & messages
⚡ Fast & lightweight (Termux compatible)

---

🚀 Commands

Command| Description
"/start"| Start the bot 💖
"/play <song>"| Play music in VC 🎶
"/pause"| Pause music ⏸
"/resume"| Resume ▶️
"/skip"| Skip current song ⏭
"/stop"| Stop & clear queue ⛔
"/queue"| Show playlist 📜

---

🛠 Installation (Termux)

1️⃣ Update & Install

pkg update && pkg upgrade -y
pkg install python git ffmpeg -y

---

2️⃣ Clone Repository

git clone https://github.com/YOUR_USERNAME/Kawaii-music-bot.git
cd Kawaii-music-bot

---

3️⃣ Install Requirements

pip install -r requirements.txt

---

4️⃣ Setup Environment Variables

Create ".env" file:

API_ID=your_api_id
API_HASH=your_api_hash
BOT_TOKEN=your_bot_token
SESSION=your_session_string

---

5️⃣ Run Bot

python bot.py

---

⚠️ Important Notes

- ❗ Make sure Assistant account joins the group
- ❗ Start voice chat manually before using "/play"
- ❗ Never expose your BOT_TOKEN or SESSION publicly
